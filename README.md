# ASECII: ASCII Sprite Editor Console II

![Preview.png](Preview.png)

Advanced ASCII painter developed as an open-source alternative to REXPaint.

- Layers
- Glyph/Foreground/Background filter
- Select
- Wand
- Move
- Delete


Image files export to ASECII JSON format, Dictionary<Point, ColoredGlyph> format, and TXT format.
